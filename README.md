ftjs 是基于 [Vue 3](https://github.com/vuejs/core) 的表单和表格组件库。

## 特性

- 📦 组件开箱即用
- 🎨 自由扩展
- 💪 强大的类型支持，所有原组件的类型提示完美继承
- 🚀 高性能的数据更新
