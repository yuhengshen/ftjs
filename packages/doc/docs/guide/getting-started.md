# 快速开始

## 特性

- 📦 开箱即用的高质量组件
- 🎨 自由去扩展组件
- 💪 强大的类型支持（包括扩展组件），所有原组件的类型提示完美继承
- 🚀 高性能的数据更新

## 环境支持

- Vue 3.3+
- TypeScript >= 5.0.0

## 安装

使用 pnpm 安装（推荐）基础库：

```bash
pnpm i tf-core
```

安装适配器（ant-design-vue）：

```bash
pnpm i tf-antd
```
