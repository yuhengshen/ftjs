# 快速开始

## 特性

- 📦 组件开箱即用
- 🎨 自由扩展
- 💪 强大的类型支持，所有原组件的类型提示完美继承
- 🚀 高性能的数据更新

## 环境支持

- Vue 3.3+
- TypeScript >= 5.0.0

## 安装

安装（推荐）基础库：

```bash
npm i @ftjs/core // [!=npm auto]
```

安装适配器（ant-design-vue）：

```bash
npm i @ftjs/antd // [!=npm auto]
```
